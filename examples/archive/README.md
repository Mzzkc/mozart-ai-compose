# Archived Example Scores

Scores in this directory are preserved for historical reference but are **no
longer recommended for current use**.

## Archived Scores

_None currently._

## Why this directory exists

If a score becomes fully subsumed by a newer score in `examples/` (with
verified feature parity), it gets moved here with a row in the table above
explaining which replacement to use instead.

## Running an archived score

```bash
mzt run examples/archive/<score>.yaml
```

Prefer the replacement unless you have a specific reason (reproducing
historical behavior, debugging a regression in the replacement).
